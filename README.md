# Dischat

A chat like discord with all basic except channels and voice chat.

## Features

- Login, Sign Up
- Create, Join, Leave, Servers
- Send and Delete Messages
- Reatime Updates

## Technologies

- React
- Firebase Auth
- Redux
- Typescript
- Material UI
- Apollo GraphQL
- Framer Motion (for animation)
